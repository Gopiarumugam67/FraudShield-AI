## 📌 Overview

FraudShield AI is an enterprise-grade Credit Card Fraud Detection platform that combines machine learning, explainable AI, and modern software engineering practices to detect fraudulent transactions in real time.

Unlike traditional fraud detection projects that stop at model training, FraudShield AI delivers a complete end-to-end solution including:

- Data preprocessing pipeline
- Feature engineering
- Machine learning model comparison
- Hyperparameter optimization
- Explainable AI (SHAP)
- REST API using FastAPI
- Interactive Streamlit dashboard
- Docker deployment
- Experiment tracking with MLflow
